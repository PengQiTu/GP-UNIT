For easy training, the dataset are suggested to structured in the following folder structures:

```
DatasetName
|--train
   |--ClassName1
      <training images>
   |--ClassName2
      <training images>
   ...
   |--ClassNameN
      <training images>   
|--test (or val)
   |--ClassName1
      <testing images>
   |--ClassName2
      <testing images>
   ...
   |--ClassNameN
      <testing images>   
```
